# MagmaTech
MagmaTech is a free c++ vulkan based engine for 3d applications.  
it features:  
texture mapping (mipmap included)  
model loader (obj files)  
texture loader (ppm files)  
shadowmap  
resolution scale  
full android support  
cubemap
